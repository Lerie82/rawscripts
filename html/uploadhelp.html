Uploading Files

If you have files created in other screenwriting programs, we make it easy to import your scripts and keep writing online.

You can upload:

  • Celtx files (.celtx)
  • Final Draft files (.fdx only)
  • Formated text files from any other software (.txt)*

*Most any software will create a formatted .txt file for you. In some programs you simply choose "Save As..." and change the format to .txt. In others, you have the option to "Export Script" as a .txt.
