Migrate Scripts

  1. Go to docs.google.com. In the left column, choose the "RawScripts" folder, then pick the scripts you want to move over to the new site.
  2. In click the "More Actions" button, choose "Export," and download all them the scripts as html files (should be default).
  3. Unzip what you have downloaded, and any scripts you downloaded should be there in files called "index.html"
  4. At RawScripts.com, upload the html files and you're done.
